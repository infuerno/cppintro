Question 3

"For" loops can always be re-written as "while" loops, and vice-versa. Are the following two programs equivalent, and what is their output? Explain your answer, and run the programs to check.

Program (a):

	#include <iostream>
	using namespace std;
						
	int main()
	{
		int count = 1;
		for (; count <= 5 ; count++)
		{
			int count = 1;
			cout << count << "\n";
		}
		return 0;
	}

Program (b):

	#include <iostream>
	using namespace std;

	int main()
	{
		int count = 1;
		while (count <= 5)
		{
			int count = 1;
			cout << count << "\n";
			count++;
		}
		return 0;
	}
